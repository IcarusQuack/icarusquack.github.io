Common CSS Properties

Properties Description Values
font-weight used to make text thinner or thicker: 100-900 (increments of 100) 400 (normal), 700 (bold)
font-style used to make text italic or normal (default) normal, italic, oblique
line-height used to set the height of a lines box. generally unitless to scale natively with the font size 1, 2.5, 3, etc
word-spacing used to increase the space inbetween each word in an element normal, .2rem, 1px, etc
letter-spacing used to increase the space inbetween each letter in an element normal, 1rem, 4px, etc
text-align used to set the alignment of the content in an element block, horizontally start, end, center, justify
a:hover use with brackets {} to add css styling to mouse hover on an element a:hover {color:red}
@font-face Local fonts/external fonts can be added with this property, followed by the custom font-family name, then source with url(location) format(type)
